# 🚦 Indian Traffic Violations Analysis

## 📌 Project Overview
End-to-end data analytics project on Indian Traffic Violations using Python, SQL, Excel & Power BI

## 🛠️ Tools Used
- **Excel** — Data Cleaning & Preparation
- **MySQL** — Data Analysis & Querying
- **Python** — Exploratory Data Analysis & Visualizations
- **Power BI** — Interactive Dashboard

## 📊 Dataset
- **Source:** Kaggle (Indian Traffic Violation)
- **Link:** https://www.kaggle.com/datasets/khushikyad001/indian-traffic-violation
- **Records:** 4,000 violations
- **Period:** 2023 - 2024
- **Region:** India (8 States)

## 🔍 Key Insights
- 🚨 Total Violations: 4,000
- 💰 Average Fine: ₹2,530
- 🏆 Top Violation: Drunk Driving (488 cases)
- 📍 Top State: Gujarat (520 violations)
- 🚗 Top Vehicle: Scooter
- ✅ Fine Paid: 49.3% (1,972 cases)
- ⚠️ Fine Unpaid: 50.7% (2,028 cases)
- 🍺 Positive Breathalyzer: 1,314 cases
